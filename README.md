## How to run

Create a virtual env:

```bash
$ mkvirtualenv --python=python3.6 checkers
```

If you already have the environment:

```bash
$ workon checkers
```

Install all dependencies: 

```bash
$ pip install -r requirements.txt
```

Run:

```bash
$ python src/checkers.py
```

Tests:
```bash
$ pytest
```
